## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/mahalakshmi-enterprises.git

cd mahalakshmi-enterprises

npm install

npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```
